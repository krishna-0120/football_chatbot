# ⚽ Football AI Expert Chatbot

A simple, Python-based chatbot powered by Google Gemini that answers any questions related to football (soccer).

## 🚀 Features
*   **Expert Knowledge:** Powered by the `gemini-2.5-flash` model.
*   **Contextual Memory:** Remembers previous questions in the conversation.
*   **Secure:** Uses environment variables to protect API keys.
*   **Topic Restricted:** Focused specifically on football.
